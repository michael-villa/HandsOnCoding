Que hace/es?

Decorator es un patrón de diseño estructural que te permite añadir funcionalidades a objetos colocando estos objetos dentro de objetos encapsuladores especiales que contienen estas funcionalidades. Esta acomoda o asigna informacion dentro de objetos anidados

Caracteristicas de calidad:

Bajo mi criterio la caracteristica de calidad mas afin a este patron es la compatibilidad, especificamente la interoperatibilidad debido a que esta es la capacidad de dos o más sistemas o componentes para intercambiar información y utilizar la información intercambiada.
